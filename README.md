GitHub Pages upload folder for Raphael Dos Santos' portfolio.

Upload the contents of this folder to the root of a GitHub repository, or place this folder's contents inside a `docs/` folder in a repository configured for GitHub Pages.

Required structure:

- `index.html`
- `portfolio-assets/`
- `.nojekyll`

If you use a repository root deployment:

1. Create a GitHub repo.
2. Upload everything in this folder.
3. In GitHub, go to `Settings > Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose your default branch and the `/ (root)` folder.

If you use a `docs/` deployment:

1. Copy all contents of this folder into a repo's `docs/` folder.
2. In `Settings > Pages`, choose the `docs/` folder as the source.
